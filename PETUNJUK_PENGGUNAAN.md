## Petunjuk untuk Admin

### 1. Login Admin
- Akses halaman login
- Masukkan email: `admin@assessment.com`
- Masukkan password: `pass123`

### 2. Manajemen Sesi Penilaian
1. **Buat Sesi Baru:**
   - Klik menu "Sesi Penilaian"
   - Klik tombol "Tambah Sesi"
   - Isi nama sesi, waktu mulai, dan durasi
   - Set status menjadi "Active"

2. **Kelola Peserta:**
   - Pilih sesi yang akan dikelola
   - Klik "Kelola Peserta"
   - Import peserta dari file CSV atau tambah manual
   - Pastikan peserta terdaftar dengan benar

### 3. Manajemen Assessment
1. **Buat Assessment:**
   - Pilih jenis assessment (Studi Kasus, In-Tray, Role-Play, LGD)
   - Untuk In-Tray, pilih mode:
     - **Mode Urutan**: Drag-drop cards (untuk sesi 1, 4, 5, 7)
     - **Mode Prioritas**: Matriks 4 kuadran (untuk sesi 6)
   - Upload file PDF jika diperlukan
   - Set durasi dan instruksi

2. **Kelola Memo (In-Tray):**
   - Buat memo untuk setiap assessment In-Tray
   - Set urutan dan status aktif
   - Pastikan memo sesuai dengan sesi penilaian

### 4. Monitoring Progress
1. **Lihat Progress Peserta:**
   - Akses menu "Progress"
   - Pilih sesi dan peserta
   - Monitor status pengerjaan
   - Download hasil assessment

2. **Lihat Matriks Prioritas:**
   - Klik tombol "Matriks In-Tray" pada progress peserta
   - Lihat kategorisasi memo berdasarkan 4 kuadran prioritas

### 5. Konfigurasi Sistem
- **Mode In-Tray per Sesi:**
  - Sesi 1, 4, 5, 7: Mode Urutan (drag-drop)
  - Sesi 6: Mode Prioritas (matriks)
- **Status Sesi:** Pastikan sesi dalam status "Active" untuk dapat diakses peserta

## Petunjuk untuk Peserta

### 1. Login Peserta
- Akses halaman login peserta
- Masukkan email dan password yang diberikan admin
- Atau gunakan PIN yang telah disediakan

### 2. Dashboard Peserta
- Lihat daftar assessment yang tersedia
- Status assessment: Belum Dimulai, Sedang Berlangsung, Selesai
- Klik "Mulai Assessment" untuk memulai

### 3. Pengerjaan Assessment

#### A. Studi Kasus
1. Baca petunjuk dan kasus yang diberikan
2. Jawab pertanyaan dalam text editor
3. Klik "Simpan Sementara" untuk menyimpan draft
4. Klik "Simpan Final" untuk menyelesaikan

#### B. In-Tray Exercise

**Mode Urutan (Drag-Drop):**
1. Baca semua memo yang tersedia
2. Drag dan drop kartu memo untuk mengatur prioritas
3. Kartu di atas = prioritas lebih tinggi
4. Klik "Lihat Detail" untuk membuka detail memo
5. Isi disposisi untuk setiap memo
6. Klik "Simpan Sementara" atau "Simpan Final"

**Mode Prioritas (Matriks):**
1. Baca semua memo yang tersedia
2. Kategorikan memo ke dalam 4 kuadran:
   - **Mendesak & Penting** (Quadrant 1)
   - **Tidak Mendesak & Penting** (Quadrant 2)
   - **Mendesak & Tidak Penting** (Quadrant 3)
   - **Tidak Mendesak & Tidak Penting** (Quadrant 4)
3. Klik tombol "Lihat Matriks" untuk melihat hasil kategorisasi
4. Isi disposisi untuk setiap memo
5. Klik "Simpan Sementara" atau "Simpan Final"

#### C. Role-Play
1. Baca skenario yang diberikan
2. Tuliskan catatan observasi dalam text editor
3. Fokus pada aspek yang dinilai
4. Simpan catatan secara berkala

#### D. LGD (Leaderless Group Discussion)
1. Baca topik diskusi
2. Tuliskan catatan observasi selama diskusi
3. Catat partisipasi dan kontribusi peserta lain
4. Simpan catatan secara berkala

### 4. Tips Pengerjaan
- **Simpan secara berkala** untuk menghindari kehilangan data
- **Baca petunjuk dengan teliti** sebelum memulai
- **Kelola waktu** sesuai durasi yang diberikan
- **Gunakan fitur auto-save** yang tersedia

## Jenis Assessment

### 1. Studi Kasus
- **Tujuan**: Mengukur kemampuan analisis dan problem solving
- **Format**: Text editor dengan pertanyaan terbuka
- **Durasi**: 60-90 menit
- **Output**: Jawaban tertulis

### 2. In-Tray Exercise
- **Tujuan**: Mengukur kemampuan manajemen prioritas dan pengambilan keputusan
- **Format**: 
  - Mode Urutan: Drag-drop cards
  - Mode Prioritas: Matriks 4 kuadran
- **Durasi**: 60-90 menit
- **Output**: Urutan prioritas dan disposisi

### 3. Role-Play
- **Tujuan**: Mengukur kemampuan komunikasi dan leadership
- **Format**: Simulasi interaksi dengan observer
- **Durasi**: 30-45 menit
- **Output**: Catatan observasi

### 4. LGD (Leaderless Group Discussion)
- **Tujuan**: Mengukur kemampuan kerja tim dan leadership
- **Format**: Diskusi kelompok tanpa pemimpin
- **Durasi**: 45-60 menit
- **Output**: Catatan observasi

## Mode In-Tray Exercise

### Mode Urutan (Drag-Drop)
- **Digunakan untuk**: Sesi 1, 4, 5, 7
- **Cara kerja**: 
  - Drag dan drop kartu memo
  - Urutan dari atas ke bawah = prioritas tinggi ke rendah
  - Auto-scroll saat drag mendekati tepi
- **Keunggulan**: Intuitif dan mudah digunakan

### Mode Prioritas (Matriks)
- **Digunakan untuk**: Sesi 6
- **Cara kerja**:
  - Kategorikan memo ke 4 kuadran
  - Berdasarkan tingkat mendesak dan penting
  - Lihat hasil dalam matriks visual
- **Keunggulan**: Analisis lebih mendalam

### Matriks Prioritas 4 Kuadran
```
┌─────────────────┬─────────────────┐
│   MENDESAK &    │ TIDAK MENDESAK &│
│    PENTING      │    PENTING      │
│   (Quadrant 1)  │  (Quadrant 2)   │
├─────────────────┼─────────────────┤
│   MENDESAK &    │ TIDAK MENDESAK &│
│ TIDAK PENTING   │ TIDAK PENTING   │
│  (Quadrant 3)   │  (Quadrant 4)   │
└─────────────────┴─────────────────┘
```

## Troubleshooting

### Masalah Umum

#### 1. Tidak Bisa Login
- **Penyebab**: Email/password salah atau akun tidak aktif
- **Solusi**: Hubungi admin untuk reset password

#### 2. Assessment Tidak Muncul
- **Penyebab**: Sesi belum aktif atau peserta belum terdaftar
- **Solusi**: Hubungi admin untuk aktivasi

#### 3. Tombol "Lihat Matriks" Tidak Muncul
- **Penyebab**: Assessment menggunakan mode urutan
- **Solusi**: Normal, hanya muncul untuk mode prioritas (sesi 6)

#### 4. Drag-Drop Tidak Bekerja
- **Penyebab**: Browser tidak mendukung atau JavaScript error
- **Solusi**: Refresh halaman atau gunakan browser terbaru

#### 5. Data Hilang
- **Penyebab**: Tidak menyimpan secara berkala
- **Solusi**: Gunakan fitur "Simpan Sementara" secara rutin
